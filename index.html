üöÄ Quick Start

Prerequisites

  • FiveM Server
  • MySQL Database
  • oxmysql Resource
  • Tebex Account (for payment processing)

Installation Steps

  1. Download TD-Store from your purchase source
  2. Extract files to your server's resources folder
  3. Import database.sql into your MySQL database
  4. Configure s_config.lua and c_config.lua
  5. Add to your server.cfg: 
    ensure oxmysql
ensure td-store
  6. Set up ACE permissions: 
    add_ace group.admin tdstore.admin allow
add_ace group.manager tdstore.manager allow

Tebex Integration

Configure your Tebex settings in s_config.lua:

SConfig.Tebex = {
    private_key = "YOUR_TEBEX_PRIVATE_KEY",
    public_token = "YOUR_TEBEX_PUBLIC_TOKEN", 
    project_id = "YOUR_TEBEX_PROJECT_ID",
    webhook_secret = "YOUR_WEBHOOK_SECRET",
    webhook = {
        url = "http://your-server-ip:30120/webhook"
    }
}

üîß Troubleshooting

Common Issues

  • Webhook Not Working - Ensure server runs on port 30120 and webhook URL is accessible
  • Database Connection Issues - Verify oxmysql loads before TD-Store
  • Permission Errors - Check ACE permissions and group assignments
  • Vehicle Spawning Issues - Verify spawn coordinates and vehicle model names
